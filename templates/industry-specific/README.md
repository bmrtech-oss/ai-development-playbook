# Industry-Specific AI Recipes

This directory contains enterprise AI recipe templates for industry-specific use cases. Each recipe emphasizes deterministic output, compliance guardrails, and token economics awareness.

- `retail-ecommerce/` — Real-time inventory orchestration and personalized style recommendations.
- `healthcare/` — HIPAA-compliant clinical note extraction patterns.
- `legal-finance/` — High-precision fact-checking using Datalog and deterministic logic.

Use these templates as a starting point for industry adoption, cross-functional architecture reviews, and secure deployment planning.
