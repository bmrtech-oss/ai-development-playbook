# Leadership Update Template

Use this format for weekly updates to stakeholders. **Signal, not noise.**

```
Subject: Engineering Update – [Week Ending YYYY-MM-DD]

## 🚀 What Shipped
- [One‑liner about a feature or improvement]
- [Another key delivery]

## 📈 Key Metrics
- SLM inference latency (p95): X ms (Δ from last week)
- Incidents: X (Y SEV1, Z SEV2)

## 🧠 Decisions Made
- [Architecture or process decision]

## 🔮 Looking Ahead
- Focus for next week: [1‑2 big rocks]

## ❓ Needs Input
- [Single clear ask from leadership]
```
