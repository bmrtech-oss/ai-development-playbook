# Awesome AI Engineering

A curated list of tools and resources referenced by the AI Platform playbook.

## LLM Evaluation

- [promptfoo](https://github.com/promptfoo/promptfoo) — Prompt evaluation and regression testing for prompt-driven AI workflows.
- [Garak](https://github.com/garak/garak) — Evaluation tooling for AI model outputs and test automation.

## RAG & Vector Databases

- [pgvector](https://github.com/pgvector/pgvector) — PostgreSQL extension for vector similarity search.
- [Chroma](https://www.trychroma.com/) — Open-source vector database for embedding and retrieval applications.

## Prompt Engineering

- [LangChain](https://github.com/langchain/langchain) — Framework for building prompt chains and retrieval pipelines.
- [Guidance](https://github.com/microsoft/guidance) — Declarative prompt engineering for structured AI workflows.

## Infrastructure as Code

- [OpenTofu](https://github.com/opentofu/opentofu) — Open-source Terraform-compatible IaC tool.
- [Terraform](https://www.terraform.io/) — Infrastructure provisioning and deployment automation.

## Testing

- [pytest](https://github.com/pytest-dev/pytest) — Python testing framework used across the backend.
- [Playwright](https://playwright.dev/) — End-to-end test automation for web and VS Code extension flows.
- [Kiwi TCMS](https://kiwitcms.org/) — Test case management for QA and release validation.

## Observability

- [Prometheus](https://prometheus.io/) — Metrics collection and alerting.
- [Grafana](https://grafana.com/) — Dashboarding and observability visualization.
- [OpenTelemetry](https://opentelemetry.io/) — Standardized tracing and telemetry for distributed systems.
